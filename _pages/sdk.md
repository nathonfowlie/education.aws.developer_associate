## Overview

- Run AWS actions without using the CLI.
- Available in multiple languages:
  - Java
  - .NET
  - Node.js
  - PHP
  - Python (boto3/botocore)
  - Go
  - Ruby
  - C++
- SDK selects us-east-1 will be set as the default region if none is specified.
